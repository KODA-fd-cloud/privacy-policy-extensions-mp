Политика конфиденциальности

Последнее обновление: 21 марта 2026 г.

КОДА — Выгрузки с МП — расширение для браузера Google Chrome, предназначенное для массового скачивания отчётов из кабинета продавца на маркетплейсах Wildberries и Ozon.

1. Какие данные собирает расширение

Расширение не собирает, не хранит и не передаёт персональные данные пользователей третьим лицам.

2. Данные, обрабатываемые локально

Для работы расширение использует следующие данные, которые обрабатываются исключительно на устройстве пользователя:

  • Cookies авторизации — используются для выполнения запросов к API маркетплейсов от имени пользователя. Расширение не читает, не копирует и не сохраняет содержимое cookies.
  • Настройки расширения — выбранные отчёты, период и параметры колонок сохраняются в локальном хранилище браузера (chrome.storage.local) для удобства пользователя.
  • Данные отчётов — скачанные отчёты временно хранятся в оперативной памяти для формирования ZIP-архива, после чего удаляются. Расширение не сохраняет данные отчётов на диск помимо итогового архива.

3. Сетевые запросы

Расширение выполняет запросы исключительно к официальным API маркетплейсов:

  • *.wildberries.ru — API Wildberries для скачивания отчётов
  • seller.ozon.ru — API Ozon для скачивания отчётов

Расширение не отправляет данные на какие-либо сторонние серверы, не использует аналитику и не содержит рекламы.

4. Разрешения

  • downloads — для сохранения ZIP-архива с отчётами на устройство пользователя
  • storage — для сохранения настроек расширения (выбор отчётов, период)
  • cookies — для авторизации запросов к API маркетплейсов
  • scripting — для выполнения запросов к API в контексте вкладки маркетплейса
  • webRequest — для определения идентификатора рекламной организации Ozon
  • Host permissions — доступ к доменам Wildberries и Ozon для скачивания отчётов

5. Безопасность

Все данные обрабатываются локально в браузере пользователя. Исходный код расширения не обфусцирован и доступен для проверки.
